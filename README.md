# 🚢 Titanic Survival Prediction

## 📌 Project Overview
This project aims to predict whether a passenger survived the Titanic disaster using Machine Learning techniques. It is a classification problem based on passenger data such as age, gender, class, and fare.

---

## 📊 Dataset
The dataset contains information about passengers:
- Age
- Sex
- Passenger Class (Pclass)
- Fare
- SibSp (siblings/spouse)
- Parch (parents/children)
- Embarked
- Survived (Target Variable)

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🔍 Steps Performed
1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Data Visualization  
4. Feature Selection  
5. Model Training (Logistic Regression)  
6. Prediction  
7. Model Evaluation  

---

## 🤖 Machine Learning Model
- Algorithm Used: Logistic Regression  
- Type: Supervised Learning (Classification)

---

## 📈 Results
- Model Accuracy: **~83%**
- Key Insights:
  - Female passengers had higher survival rates
  - First-class passengers survived more
  - Third-class passengers had lower survival chances
